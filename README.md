# DSA-Practice

This is a TypeScript starter project for learning Data Structures & Algorithms (DSA).

Quick setup

1. Install dependencies:

	npm install

2. Development (auto-restart):

	npm run dev

3. Build:

	npm run build

4. Run compiled code (after build):

	node dist/index.js

Project layout

- `src/` — TypeScript source files
  - `data-structures/` — implementations (linked list example included)
  - `index.ts` — small demo runner

Next steps

- Run `npm install` in this folder to install TypeScript and helpers.
- Add more data structure implementations under `src/data-structures/`.
- Add tests (recommended: vitest or jest + ts-jest) when comfortable.
